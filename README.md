
# Setup and run instructions.
## Installation Guide
Open terminal in folder, connect mongodb in background.
```terminal
npm install
npm run dev
```

## check git ignore 
-check git ignore file and load the dependencies

## .env Configuration
```
MONGO_URI=
JWT_SECRET=
PORT=9997
OPENAI_API_KEY=
OPENAI_MODEL=gpt-3.5-turbo
```
